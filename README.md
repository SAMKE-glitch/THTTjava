## How to Compile & Run

Open Ubuntu terminal in the project folder.

### Compile Java File
```bash
javac <FileName>.java
```

Run Compiled File
```bash
java <FileName>
```

Example:

```bash
javac Question1_GetLongestString.java
java Question1_GetLongestString
```
